Delveline excels as a security analysis tool, offering the ability to detect vulnerabilities such as:

Detection is aligned with OWASP ASVS, CWE, and OSV.dev standards. OSV.dev, backed by Google, enhances detection of known vulnerabilities (CVEs) in project dependencies.

Industry Standards and Scoring

Delveline integrates CWE (Common Weakness Enumeration) for vulnerability classification and uses CVSS (Common Vulnerability Scoring System) for severity scoring.

With OSV.dev integration, Delveline can identify outdated or vulnerable dependencies in your project, adding an extra layer of security.

Runtime-Assisted Static Analysis

Delveline employs a unique runtime-assisted static analysis approach. This allows static analysis to be supplemented by runtime data, making it more adaptive and effective than purely static tools.

Target Audience

Delveline is designed for projects requiring deep security analysis, especially those aiming to ensure compliance with industry standards.
